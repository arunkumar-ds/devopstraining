1. Create a directory
mkdir my_folder

2. Move into the directory
cd my_folder

3. Create my_file.txt with some text
echo "This is my file." > my_file.txt

4. Create another_file.txt with some text
echo "This is another file." > another_file.txt

5. Concatenate another_file.txt into my_file.txt
cat another_file.txt >> my_file.txt

6. Display the updated content of my_file.txt
cat my_file.txt

7. List all files and directories
ls -l

Create 20 .txt files
for i in {1..20}; do
    echo "Sample text for file $i" > file_$i.txt
done

Rename the first 5 .txt files to .yml

Run:

for i in {1..5}; do
    mv file_$i.txt file_$i.yml
done

Print the latest created top 5 files

This lists the newest 5 files based on modification time:

ls -lt | head -5
